# bookCLUB

اسکلت پروژه‌ی Qt/C++ کلاینت-سرور برای سیستم کتاب الکترونیکی.

این بسته فقط ساختار فایل‌ها را آماده می‌کند:
- بخش مشترک (common)
- بخش سرور (server)
- بخش کلاینت (client)
- منابع، UI، QML، دیتابیس، تست، مستندات و نصب

برای توسعه‌ی واقعی، فایل‌های .cpp و منطق برنامه باید بر اساس این اسکلت تکمیل شوند.
