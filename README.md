# BookClub

BookClub is a Client-Server digital library system developed with

- Qt 6
- C++
- CMake
- TCP Socket
- SQLite

## Architecture

- Client
- Server
- Shared(Common)

## Roles

- User
- Publisher
- Admin

## Features

- Authentication
- Book Store
- Cart
- Library
- Notifications
- PDF Reader

## Technologies

- Qt Widgets
- Qt Network
- Qt SQL
- CMake
